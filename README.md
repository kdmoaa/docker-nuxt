# nuxt-docker

## Build Setup

### Docker
```bash
# you should in development/ or staging/ or production/
$ docker-compose build
$ docker-compose up -d
$ docker-compose exec web bash

```
### Nuxt.js
``` bash
# install dependencies
$ yarn install

# build project
$ yarn run build

# start project
$ yarn start
```

you can access by 'localhost:3000'
